Sunscape merupakan aplikasi pemutar musik yang dikembangkan dengan bahasa pemrograman Python.  Nama "Sunscape" terinspirasi dari perpaduan "sunset" dan "soundscape", merefleksikan filosofi aplikasi yang menghadirkan pengalaman mendengarkan musik yang indah dan menyenangkan layaknya menikmati senja.  GUI ini menampilkan antarmuka grafis menggunakan library Tkinter, sedangkan untuk pemutaran audio menggunakan library Pygame.  

GUI ini menerapkan pembagian hak akses menjadi dua peran utama yakni Admin dan User. Peran admin bertugas mengelola database lagu, sedangkan peran user dapat menikmati musik serta mengatur playlist pribadi. Pembagian peran ini memungkinkan pengelolaan data yang  sesuai dengan fungsi masing-masing. 

Pada bagian inti sistem, Sunscape mengimplementasikan lima struktur data berbeda yang bekerja secara bersama-sama. Doubly Linked List berfungsi sebagai penyimpanan utama untuk daftar lagu dan playlist, Hash Map digunakan untuk pencarian cepat berdasarkan ID lagu, Queue mengatur antrian lagu yang akan diputar, Stack menyimpan riwayat pemutaran, dan struktur Record merepresentasikan data lengkap setiap lagu. Kombinasi ini menghasilkan sistem yang efisien dengan performa optimal. 

Fitur utama aplikasi meliputi fungsi CRUD (Create, Read, Update, Delete) lengkap pada admin mode, sedangkan pada user mode tersedia sistem pemutaran musik dengan berbagai kontrol, manajemen playlist yang fleksibel, algoritma pencarian yang toleran terhadap kesalahan ketik, dan sistem rekomendasi lagu serupa berdasarkan kemiripan artis dan genre. Selain itu, Sunscape memiliki mekanisme sinkronisasi data otomatis dimana setiap perubahan data oleh admin langsung berdampak pada user. 

Dari segi antarmuka, aplikasi ini menggunakan tema warna gradasi sunset dengan tampilan yang modern dan menarik. Struktur kode program diorganisir secara modular dengan pemisahan yang jelas antara struktur data dan tampilan antarmuka. Melalui Sunscape, seluruh materi struktur data yang telah dipelajari dapat diterapkan secara praktis dalam membangun GUI pemutar musik yang fungsional.
